# Sistema Avanzado de Gestión de Inventario

**Autor:** Leonardo Benjamín Córdova Jaramillo  
**Universidad:** Universidad Estatal Amazónica - UEA 
**Materia:** Programación Orientada a Objetos (POO) en Python  
**IDE utilizado:** PyCharm  
**Control de versiones:** Git y GitHub  

Este proyecto implementa un **sistema avanzado de gestión de inventario** para una tienda, aplicando los conceptos de **Programación Orientada a Objetos (POO)**, usando **colecciones de Python** y **archivos de texto** para la persistencia de datos.

La tarea corresponde a la actividad:

> **“Sistema Avanzado de Gestión de Inventario”**  
> - Aplicar POO para la estructura del programa.  
> - Utilizar colecciones (listas, diccionarios, conjuntos, tuplas) para gestionar el inventario.  
> - Implementar lectura y escritura de archivos para el almacenamiento persistente.  
> - Crear un menú en consola para gestionar el inventario.
